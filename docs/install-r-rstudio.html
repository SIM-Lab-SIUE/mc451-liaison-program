Install R & RStudio

Why R & RStudio?

R is the industry standard for statistical analysis, data visualization, and reproducible research in mass communications. RStudio is the professional interface that makes R accessible and powerful. Together, they‚Äôre essential tools for any Communication Liaison who works with data.

In this course, you‚Äôll use R to: - Clean and analyze data from surveys and content analysis - Create publication-ready visualizations and reports - Build reproducible workflows for your research - Collaborate on data analysis projects

What You‚Äôll Install

  1. R ‚Äî The programming language (free, open-source)
  2. RStudio ‚Äî The professional development environment (free, open-source)

This is a two-step process: install R first, then RStudio.

Step 1: Install R

For Windows

  1. Go to r-project.org
  2. Click ‚ÄúDownload R‚Äù in the left menu
  3. Click the ‚ÄúDownload R for Windows‚Äù link
  4. Click ‚Äúbase‚Äù

    • Accept the default installation location

  1. Go to r-project.org
  2. Click ‚ÄúDownload R‚Äù in the left menu

  5. Run the installer (.pkg file)

For Linux (Ubuntu/Debian)

To verify: Type R --version in Terminal.

Step 2: Install RStudio

For Windows, macOS, or Linux

  1. Go to posit.co/download/rstudio-desktop/
  2. Scroll down to ‚ÄúAll Installers‚Äù or ‚ÄúDownload RStudio Desktop‚Äù
  3. Select your operating system:
    • Windows: Download the .exe file
    • macOS: Download the .dmg file
    • Linux: Download the appropriate .deb or .rpm file
  4. Run the installer
    • Windows: Click through all default options
    • Mac: Drag RStudio to Applications folder
    • Linux: Follow the package manager prompts
  5. Open RStudio (you should see R is installed when you open it)

Step 3: Verify RStudio Works

  1. Open RStudio (should be in your Applications/Programs menu)
  2. You should see four panes:
    • Top left: Script editor (blank)
    • Bottom left: Console (shows > prompt)
    • Top right: Environment & History
    • Bottom right: Files, Plots, Packages, Help
  3. In the Console (bottom left), type this command and press Enter:
print("Hello, Communication Liaison!")
  1. You should see the output:
[1] "Hello, Communication Liaison!"

If you see this output, you‚Äôre ready to go!

Step 4: Install Key Packages

RStudio comes with base R, but you‚Äôll need additional packages for data analysis. Open RStudio and copy these commands into the Console one at a time:

For Data Analysis & Visualization

This installs a collection of packages including: - ggplot2 ‚Äî Create professional visualizations - dplyr ‚Äî Clean and transform data - readr ‚Äî Import data from CSV, Excel, etc. - tidyr ‚Äî Reshape data

Wait for it to finish (you‚Äôll see a blank > prompt when done)

For Survey Analysis

Used for analyzing Likert-scale survey responses.

For Reporting

install.packages("rmarkdown")

Creates professional PDF and HTML reports from your analysis.

Optional: Statistics Packages

For later phases, you might want these too:

Step 5: Create Your First R Project

RStudio projects help you organize your work. Let‚Äôs create one:

  1. In RStudio, click File ‚Üí New Project
  2. Select New Directory ‚Üí New Project
  3. Name your project mc451-research or similar
  4. Choose where to save it (suggest Documents)
  5. Click Create Project

You should now see your project name in the top-right corner of RStudio.

Step 6: Test Your Setup with Data

Let‚Äôs make sure everything works by loading a dataset:

In the Console, type:

You should see a table with car data. If this works, your R setup is complete!

Understanding the RStudio Interface

Area Purpose
Top Left (Script Editor) Write R code here and save as .R files
Bottom Left (Console) Results appear here; you can type commands directly
Top Right (Environment) Shows all variables and data you‚Äôve loaded
Bottom Right (Files/Plots/Help) Navigate files, view plots, access documentation

Basic RStudio Workflow

  1. Create a new script: File ‚Üí New File ‚Üí R Script
  2. Write code: Type your commands in the script
  3. Run code: Select lines and press Ctrl+Enter (Windows/Linux) or Cmd+Enter (Mac)
  4. See results: Output appears in the Console or Plots pane
  5. Save your work: Ctrl+S or File ‚Üí Save

Common First-Time Issues

‚ÄúCommand not found: R‚Äù

Problem: Your computer doesn‚Äôt recognize the R command.

Solution: - Try restarting your computer after installation - Check that R installed correctly (reinstall if needed) - On Mac/Linux, you may need to use /usr/bin/R instead

RStudio Opens But Doesn‚Äôt Find R

Problem: RStudio says ‚ÄúR not found‚Äù when you open it.

Solution: 1. Make sure you installed R first (before RStudio) 2. In RStudio, go to Tools ‚Üí Global Options ‚Üí General 3. Under ‚ÄúR sessions‚Äù, check the ‚ÄúR home directory‚Äù 4. If empty, browse to your R installation folder and select it 5. Click ‚ÄúOK‚Äù

‚ÄúError: Could not find function‚Äù

Problem: You get an error like could not find function "ggplot"

Solution: You need to load the package first. Run:

Installation Stuck or Takes Forever

Problem: install.packages() command seems frozen.

Solution: 1. Press Escape to stop it 2. Try installing from a different CRAN mirror:

‚ÄúPermission denied‚Äù Error (Mac/Linux)

Problem: Error when installing packages.

Solution: Try:
